Week 12 Notes

Publishing to GitHub

You can use Glitch to publish directly to GitHub. Use the "Project Info" menu in the top-left (should show the name of your project) and select "Advanced Options". From there, click "Grant repo access" to allow Glitch to access your GitHub repositories. Then press Export to GitHub. You will be prompted to enter your username and repository name, be sure these are typed exactly as they appear in GitHub.

From there you should be able to merge your changes directly on the GitHub website.

Be sure your repository contains a file named CNAME. Your CNAME file should include only your domain name on a sigle line.

Registering a Domain Name

Domain names can be registered with a variety of providers. A few recommendations include:

  • Google Domains
  • Namecheap
  • Porkbun
  • Freenom (offers free .tk/.ml/.ga domain names!)

Pointing to GitHub

Once you have registered your domain name, use your domain provider's control panel to set A records which point to:

  • 192.30.252.153
  • 192.30.252.154
